**10/22**

나는 폰트를 만들기로 하였다.

font의 컨셉은 glitch 효과 이다.

**글리치 효과** : **버그가 발생**하면 화면이 지직거리는 
일이 많기 때문에 **이런 지직거리는 화면**을 
“글리치 스크린(화면)”이라고 부르기도 합니다. 
이것이 이미지 편집에서는 “글리치 스크린”을
의도적으로 발생시키는 이펙트를 지칭하는 
”글리치” 효과로 불리고 있습니다.
